  1. Enter the minimum sensitivity value in MinSens
  2. Enter the maximum sensitivity value in MaxSens
  3. Try the sensitivity displayed on Sens
    • Tap Lower if you want to lower the sensitivity
    • Tap Higher if you want to increase the sensitivity
    • Tap Undo if you want to try the previous sensitivity again
  4. Repeat step 3
  1. MinSensに設定できる感度の最小値を入力
  2. MaxSensに設定できる感度の最大値を入力
  3. Sensに表示された感度を試す
    • 感度を下げたい場合、Lowerを選ぶ
    • 感度を上げたい場合、Higherを選ぶ
    • 1つ前の感度をもう一度試したい場合、Undoを選ぶ
  4. 3を繰り返す
